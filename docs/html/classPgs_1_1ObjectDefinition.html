uint32_t ObjectDefinition::read3Bytes ( const uint8_t *  data,
uint16_t &  readPos 
)
inlinestaticprotected

Reads 3 bytes of data from the provided data.

The number of bytes used to store an object data segment is held as a 24-bit value in the PGS file. A 32-bit value is the next closest thing we can use on an average computer, so the value is cast to 32-bits and 0-padded.

readPosposition in data array to read from
